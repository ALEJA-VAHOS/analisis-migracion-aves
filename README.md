#  Análisis de Migración de Aves

Este proyecto analiza patrones de migración de aves utilizando técnicas de análisis de datos y machine learning.

## Tecnologías utilizadas
- Python
- pandas
- numpy
- matplotlib
- seaborn
- prince (FAMD)
- scikit-learn

##  Archivos del proyecto
- Bird migration dataset.xls → dataset original
- srcanalisis.py → código del análisis

##  Funcionalidades
- Limpieza de datos
- Transformación de variables
- Análisis exploratorio
- Reducción de dimensiones (FAMD)
- Clustering con KMeans
